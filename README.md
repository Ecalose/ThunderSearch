# ThunderSearch 闪电搜索器

利用ZoomEye的官方api，结合[开发文档](https://www.zoomeye.org/doc#user)，做了这么一个GUI界面的搜索器。目前支持查询`host_search`

- 支持本地access_token的读取(免登陆)
- 支持账号(邮箱)+密码登录
- 查询显示结果仅为部分，完整内容保存至`指定的文件`
- 支持查询当月剩余额度

---
## 使用
```
pip install -r requirements.txt
```
